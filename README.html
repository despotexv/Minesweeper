Minesweeper Game

This Minesweeper game is a Python implementation of the classic puzzle game. The game presents a grid of tiles, some of which contain hidden mines. The objective is to clear the grid without detonating any mines.

Gameplay

  • The game generates a grid with hidden mines.
  • Clicking on a tile will either reveal the number of mines in adjacent tiles or a mine, ending the game.
  • The player can flag tiles they suspect contain mines to avoid accidentally clicking them.
  • The game is won when all non-mine tiles are revealed.

Features

  • Dynamic grid generation based on user-defined size and mine probability.
  • Safe first click to ensure the first tile revealed is never a mine.
  • Recursive reveal of adjacent non-mine tiles for faster board clearing.
  • Simple ASCII-based UI for easy gameplay.

How to Play

  1. Run the game in a Python environment.
  2. Enter the desired grid size and mine probability when prompted.
  3. Use mouse clicks to reveal tiles and right-click to flag potential mines.
  4. Use logic and caution to reveal all safe tiles!

Requirements

  • Python 3.x
  • No external libraries are required for the console version.
  • Pygame library required for the graphical version.

Installation

Clone or download this repository. If you're playing the graphical version, ensure Pygame is installed:


pip install pygame

Run the game using Python :


python minesweeper.py Length Width Probability
